# MG Custom App

**Customization for Frappe Framework**

## Features

- **Enhanced UX/UI**: A premium, modern theme for the Frappe Desk.
- **Custom Scripts**: Global Javascript utilities for better interactivity.
- **Frappe Cloud Ready**: Structured to be easily deployed on Frappe Cloud.

## Installation

Get the app from your Frappe Cloud dashboard or install manually:

```bash
bench get-app mg https://github.com/yourusername/mg
bench --site [your-site] install-app mg
```

## Customization

The theme follows a modern design system with:
- **Primary Color**: Modern Blue (#2563eb)
- **Typography**: Inter font family
- **Components**: Rounded corners, soft shadows, and clean layouts.

Modify `mg/public/css/mg.css` to tweak the visual styles.
